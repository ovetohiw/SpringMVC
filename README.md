# SpringMVC

## Задачи проекта
* создать web-приложение сохраняющее работников в базу данных
* поля для работников
    * Имя;
    * Фамилия;
    * Зарплата;
    * Департамент.


# В проекте используется:
* Hibernate - для подключения к MySQL;
* Spring ORM;
* JSTL;
* Javax persistance;
* Maven;
* Spring.
